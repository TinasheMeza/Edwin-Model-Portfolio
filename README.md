# Edwin Ndifon - Fashion Model Portfolio

A modern, futuristic portfolio website for fashion model Edwin Ndifon, featuring glassmorphism UI, smooth animations, and interactive elements.

## Features

- **Hero Section** - Full-screen hero with animated text and glassmorphism overlay
- **Portfolio Grid** - Interactive image grid with parallax hover effects
- **Campaigns Section** - Expandable campaign cards with smooth animations
- **About Section** - Professional bio with editorial-style layout
- **Contact Section** - Contact form and direct contact information
- **Image Modal** - Lightbox for viewing portfolio images in full size

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Modern ES6+ JavaScript

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Design Features

- **Glassmorphism UI** - Frosted glass effects throughout
- **Dark Theme** - Black, charcoal, and warm grey color palette
- **No Blue/Purple** - Strictly neutral tones as per requirements
- **Mobile-First** - Fully responsive design
- **Smooth Animations** - Framer Motion powered transitions
- **Parallax Effects** - Interactive hover effects on portfolio images

## Customization

### Replace Placeholder Images

Update the image URLs in:
- `src/components/Portfolio.jsx` - Portfolio grid images
- `src/components/Campaigns.jsx` - Campaign cover images

### Update Content

- **About Section**: Edit `src/components/About.jsx`
- **Campaigns**: Update campaign data in `src/components/Campaigns.jsx`
- **Contact Email**: Already set to `edwinndifon5@icloud.com`
- **Instagram**: Already linked to `@edwin.ndifon`
